# Amazon-Sales-Analysis-Germany
E-commerce Sales Analysis (Amazon Germany 2022–2024)
## 📊 Project Preview

![Sales Trend](images/sales_trend.png)
# Dataset Information

This project uses Amazon Germany e-commerce sales data from 2022–2024.

## Features Included

* Product Category
* Revenue
* Units Sold
* Price
* Customer Ratings
* Region


## 📂 Project Structure

* data/ → dataset info
* notebooks/ → Jupyter analysis
* reports/ → final report
* images/ → charts
* src/ → scripts

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Power BI

## 📊 Key Insights

* Electronics category dominates revenue
* Beauty & Sports categories show high growth
* Books category declining
* Price impacts sales differently across categories

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt
2. Run notebook in Jupyter
